# 小儿推拿门店拓客小程序 - 产品规划

> **定位**：这是一个**拓客系统**，不是商城系统。所有功能围绕"帮门店获客"设计。
> 商城只是工具，裂变才是灵魂。

---

## 一、核心裂变逻辑

```
用户 A 花 19.9 元购买「祛湿药浴体验装」
    ↓ 分享专属链接
好友 B 通过链接购买 → A 立返 15.9 元 💰
好友 C 通过链接购买 → A 再返 15.9 元 💰
好友 D 通过链接购买 → A 再返 15.9 元 💰
    ↓ 邀请越多赚越多，疯狂分享！
门店短期内卖爆产品 + 获得大量新客户 🚀
```

**规则**：一级返现、多劳多得、每位好友限购 1 份、合规安全。

## 二、功能架构（极简）

### C 端小程序（4 个核心页面）

| 页面 | 功能 | 拓客价值 |
|---|---|---|
| **首页** | 门店信息 + 裂变活动 + AI 入口 | 流量承接 |
| **AI 看舌象** | 拍照→AI 分析→报告→产品推荐→分享 | 吸引 + 裂变 |
| **商城** | 产品/服务展示、下单、套餐核销 | 交易闭环 |
| **我的** | 订单、返现、邀请、舌象历史 | 用户留存 |

> 打开后台第一眼 = **今天拉了多少新客**

| 模块 | 功能 |
|---|---|
| **拓客大盘**（首页） | 新客数、裂变订单、返现金额、转化率 |
| 裂变活动 | 创建/管理裂变商品、设置返现金额 |
| 商品管理 | 产品/服务/套餐 CRUD、上下架 |
| 套餐 & 核销 | 创建套餐、扫码核销 |
| AI 舌象设置 | API 配置、提示词、额度 |
| 门店设置 | 名称、Logo、地址、Banner |
| 订单管理 | 订单列表、退款处理 |
| 支付配置 | 微信支付商户号配置 |

## 三、核心功能详细设计

### 3.1 🚀 裂变返现

**门店后台配置**：
- 选择商品 → 设活动价（¥19.9）→ 设返现金额（¥15.9）
- 每位新用户限购 1 份、活动总库存、起止时间
- 返现到小程序余额（可用于购买其他商品）

**C 端流程**：购买 → 获得专属分享码 → 分享 → 好友购买 → 立返现金

**多劳多得**：用户 A 可邀请不限数量的好友，每个好友购买都返 ¥15.9，邀请越多赚越多。

### 3.2 🔮 AI 看舌象

- 拍照指引 → 上传 → 云函数转发门店配置的 AI API → 报告
- 报告：舌质结论 + 特征 + 调理建议 + 产品推荐
- 分享报告生成卡片（含门店小程序码）→ 裂变
- 门店后台自行配置 API（地址、密钥、模型、提示词）
- 每日额度按会员等级配置

### 3.3 🛒 极轻量商城

> 自建订单系统 + **微信支付 API**（门店配置自己的商户号）

**商品类型**：

| 类型 | 示例 | 交付方式 |
|---|---|---|
| 实物产品 | 药浴包、敷贴、精油 | 快递 / 到店自提 |
| 单次服务 | 小儿推拿体验 | 到店核销 |
| 套餐服务 | 春季助长套餐 | 多次到店核销 |

**套餐核销**：
```
春季助长套餐 ¥699 → 泡浴×5 + 推拿×6 + 敷贴×9
购买后生成核销码，每次到店扫码核销一次
```

### 3.4 地推场景

```
扫码 → 免费 AI 看舌象 → 报告推荐产品 → ¥19.9 下单
→ 获得分享返现资格 → 回去分享给宝妈群 → 裂变开始 🚀
```

## 四、技术架构

| 层级 | 方案 |
|---|---|
| C 端 | 微信小程序原生 |
| 后端 | **微信云开发**（云函数 + 云数据库 + 云存储） |
| 支付 | **微信支付 API**（每个门店配置自己的商户号） |
| AI | 门店自行配置 AI API（兼容 OpenAI 协议） |
| 管理后台 | 轻量 Web 页面 |

### 云数据库 Collections

```
stores              门店信息
ai_config           AI 舌象配置
products            商品（实物/服务/套餐）
packages            套餐明细（服务项×次数）
package_usage       套餐核销记录
orders              订单
order_items         订单明细
fission_campaigns   裂变活动
fission_records     裂变返现记录
tongue_reports      舌象报告
users               用户信息（含邀请关系）
pay_config          微信支付商户号配置
```

### 项目目录

```
裂变小程序/
├── miniapp/                    # 微信小程序

│   │   ├── tongueAnalysis/     # AI 舌象分析（透传 AI 配置）

│   │   └── payApi/             # 支付回调、支付下单、退款

│   ├── app.json
│   ├── app.js
```
