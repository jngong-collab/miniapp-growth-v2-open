# 精油系列产品说明

---

## 1. 鼻舒保健油

• **调理方向**：偏重祛风散寒、宣通鼻窍、清热燥湿，缓解鼻塞、鼻痒、流涕等鼻部不适。

• **配方思路**：以苍耳子、辛夷、鹅不食草、白芷为核心宣通鼻窍，配合金银花、黄芩、石菖蒲、野菊花清热燥湿，冰片、薄荷脑开窍醒神，麻油为基质调和诸药。整体属于"通窍宣肺 + 祛风清热"的外用滴涂思路。

• **适合人群**：
1. 风寒、感冒、鼻炎导致的鼻塞、鼻堵、鼻痒、鼻干、流鼻涕、打喷嚏等局部不适的亚健康人群
2. 鼻腔不通、遇冷加重的儿童及成人
3. 需要外用辅助调理鼻部症状者

• **体质参考**：多见于风寒束表、鼻窍不利或风热上扰型体质，常见鼻塞流涕、嗅觉减退。

• **使用提示**：
1. 外用。先用温水清洁鼻腔，取本品适量滴于（或喷于）鼻腔即可，建议每日使用 2-3 次
2. 本品为外用品，禁止食用，应放置于儿童不易接触的地方，避免误食
3. 如使用本品期间出现红肿、瘙痒等不适，请立即停用或就医
4. 本品为保健用品，不代替药品、医疗器械、消毒用品、保健食品、化妆品等使用

## 2. 脾胃舒保健精油

• **调理方向**：偏重温中健脾、理气消胀、和胃止痛，缓解腹部冷痛、食欲不振、积食腹胀。

• **配方思路**：以焦山楂、陈皮、砂仁、木香为核心理气健脾、消食化积，配合肉桂、制厚朴、苍术温中燥湿，甘草、茯苓健脾益气，丁香温胃降逆，辅以甜橙精油、生姜精油、佛手柑精油芳香行气。整体属于"温中理气 + 健脾消胀"的外用按摩思路。

• **适合人群**：
1. 腹部冷痛、食欲不振、恶心反酸、积食腹胀、嗳气频繁等引起身体局部不适的亚健康人群
2. 脾胃虚寒、消化不良的儿童及成人
3. 饮食积滞、脾胃运化偏弱需要外用辅助调理者

• **体质参考**：多见于脾胃虚寒或脾胃气滞型体质，常见腹部喜温喜按、舌苔白腻、食欲差。

• **使用提示**：
1. 外用。清洁穴位皮肤，取本品适量涂于神阙穴或其他相应穴位按摩至吸收，建议每日使用 1-2 次
2. 本品为外用品，禁止食用，应放置于儿童不易接触的地方，避免误食
3. 如使用本品期间出现红肿、瘙痒等不适，请立即停用或就医
4. 本品为保健用品，不代替药品、医疗器械、消毒用品、保健食品、化妆品等使用
