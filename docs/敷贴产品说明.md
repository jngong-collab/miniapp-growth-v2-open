# 靶向敷贴产品说明

---

## 1. 小儿咳喘贴

• **调理方向**：偏重止咳化痰、宣肺平喘，缓解咳嗽、咳痰引起的身体不适。

• **配方思路**：以百合、款冬花、甘草、黄芩、大黄、川贝母、射干、苏子、刘寄奴、没药、冰片、蜂蜜为原料，经加工制成的小儿咳喘保健贴。整体属于"外贴止咳 + 化痰宣肺"的组合思路。

• **适合人群**：
1. 咳嗽反复、咳痰不畅的亚健康人群
2. 咳喘引起身体不适者
3. 需要外用贴敷辅助调理咳嗽的儿童

• **体质参考**：适用于咳嗽、咳痰引起的身体不适的亚健康人群，通过外用贴敷相应穴位，具有促进康复和健康的作用。

• **使用提示**：
1. 本品为外用品，禁止食用，应放置于儿童不易接触的地方，避免误食
2. 婴幼儿必须在成人监护下使用，贴敷后不见好转，请立即就医
3. 如使用本品期间出现红肿、瘙痒等不适，请立即停用或就医
4. 本品为保健用品，不代替药品、医疗器械、消毒用品、保健食品、化妆品等使用

## 2. 小儿鼻炎贴

• **调理方向**：偏重通鼻窍、散风寒，缓解鼻塞、鼻痒、流涕等鼻部不适。

• **配方思路**：以辛夷、鹅不食草、苍耳子、白芷、广藿香、苦杏仁、人工麝香、蜂蜜为原料，经加工制成的小儿鼻炎保健贴。整体属于"通窍散寒 + 疏风清热"的组合思路。

• **适合人群**：
1. 鼻腔不适导致的鼻塞、鼻堵、鼻痒、鼻干、流涕的亚健康人群
2. 鼻炎反复发作者
3. 遇冷加重、清涕较多的儿童

• **体质参考**：以鼻塞、清涕、遇冷加重、反复发作为主，寒湿型鼻窍不利。

• **使用提示**：
1. 本品为外用品，清洁贴敷部位，将本品贴敷于双侧迎香穴、鼻通穴或印堂穴，建议8-12小时更换一次
2. 婴幼儿必须在成人监护下使用，贴敷后不见好转，请立即就医
3. 如使用本品期间出现红肿、瘙痒等不适，请立即停用或就医
4. 本品为保健用品，不代替药品、医疗器械、消毒用品、保健食品、化妆品等使用

## 3. 儿童腺样体贴

• **调理方向**：偏重清热通下、活血凉血、解毒利咽，缓解腺样体肥大、扁桃体及咽喉不适引起的局部不适。

• **配方思路**：以牛蒡子、玄参、连翘、大黄、黄连、苦地丁、牡蛎、黄芪、细辛、防风、薄荷、冰片、蜂蜜为原料，经加工制成的儿童腺样体保健贴。整体属于"清热解毒 + 活血散结"的组合思路。

• **适合人群**：
1. 腺样体、扁桃体、咽喉不适引起的腺样体肥大、红肿、肿痛、鼻塞、打鼾等局部不适的亚健康人群
2. 上焦热毒偏盛、痰湿结聚体质的儿童

• **体质参考**：常见热毒蕴结或痰瘀互结表现，舌质偏红，苔黄或黄腻较常见。

• **使用提示**：
1. 外用，清洁贴敷部位，将本品贴敷于下颚处（双侧）、涌泉穴（双侧）或相应穴位上，建议8-12小时更换一次
2. 婴幼儿必须在成人监护下使用，贴敷后不见好转，请立即就医
3. 如使用本品期间出现红肿、瘙痒等不适，请立即停用或就医
4. 本品为保健用品，不代替药品、医疗器械、消毒用品、保健食品、化妆品等使用

## 4. 小儿胀气贴

• **调理方向**：偏重健脾和胃、理气消胀，缓解脾胃不适、肠道胀气、泄泻、腹痛。

• **配方思路**：以肉桂、炒丁香、炒谷芽、党参、小茴香、木香、藿香、香附、炙甘草、砂仁、徐长卿、当归、苏叶、蜂蜜为原料，经加工制成的小儿胀气保健贴。整体属于"温中理气 + 健脾消胀"的组合思路。

• **适合人群**：
1. 脾胃不适、肠道胀气、泄泻、腹痛引起身体不适的亚健康人群
2. 吃后腹胀、消化不良的儿童
3. 腹部受凉后不适者

• **体质参考**：多见脾胃虚寒或气滞型表现，腹部胀满、食欲差。

• **使用提示**：
1. 外用，清洁贴敷部位，将本品贴敷于神阙穴、涌泉穴或相应穴位上，建议8-12小时更换一次
2. 婴幼儿必须在成人监护下使用，贴敷后不见好转，请立即就医
3. 如使用本品期间出现红肿、瘙痒等不适，请立即停用或就医
4. 本品为保健用品，不代替药品、医疗器械、消毒用品、保健食品、化妆品等使用
