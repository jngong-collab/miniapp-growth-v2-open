# 上线操作清单

## 1. 微信支付申请与取件

- **登录地址：** `pay.weixin.qq.com` -> 扫码登录。
- **获取商户号 `MCH_ID`：**
  登录后 -> 鼠标移到右上角头像 -> 记录那串数字。
- **获取 `API_V3_KEY`：**
  账户中心 -> API 安全 -> 找到【API v3 密钥】 -> 点击“设置” -> 自己编一个 **32 位密码** -> 保存到安全的地方。
- **获取 API 证书（用于退款）：**
  账户中心 -> API 安全 -> 找到【API 证书】 -> 点击“申请证书” -> 下载压缩包。
- **解压压缩包后确认这两个东西：**
  - 文件：`apiclient_key.pem`
  - 证书序列号：在商户平台证书列表里查看并记录
- **保存要求：**
  把 `MCH_ID`、`API_V3_KEY`、`apiclient_key.pem` 放到安全的电脑或加密文档里，不要随便发群里。

## 2. 代码上传与数据库初始化

- 打开项目文件夹下的命令行 / 终端。
- 输入下面这条命令并回车：

```bash
npm run deploy:cloud
```

- 等待执行完成。
- 再输入下面这条命令并回车：

```bash
npm run db:init
```

- 等待执行完成。

## 3. 云端填参数

- **登录地址：** 腾讯云开发 CloudBase 控制台。
- 进入你的环境。
- 左侧菜单找到【云函数】。
- 找到 `payApi` 函数。
- 点击“配置”或“高级设置”。
- 找到【环境变量】。
- 依次添加下面 3 个变量：

| 键 | 值 |
| --- | --- |
| `MCH_ID` | 填刚才记录的商户号 |
| `API_V3_KEY` | 填刚才设置的 32 位密码 |
| `WECHAT_PAY_PRIVATE_KEY` | 用记事本打开 `apiclient_key.pem`，把里面全部内容复制进去 |

- 如果页面还要求填写证书序列号，再加一条：

| 键 | 值 |
| --- | --- |
| `CERT_SERIAL_NO` | 填刚才记录的证书序列号 |

- 点击“保存”或“重新部署”。

## 4. 一分钱测试

- 打开微信开发者工具。
- 点“预览”。
- 用手机微信扫码。
- 在小程序里找一个商品。
- 把价格改成 `0.01` 元，或者新建一个 `0.01` 元测试商品。
- 用自己的微信完成支付 `0.01` 元。
- 支付成功后，确认：
  - 微信扣款成功
  - 小程序订单变成“已支付”
- 再进入订单页面或后台。
- 点击“退款”或“申请退款”。
- 退款成功后，确认：
  - 微信收到退款
  - 订单变成“已退款”

## 5. 最后核对

- `MCH_ID` 已拿到。
- `API_V3_KEY` 已拿到。
- `apiclient_key.pem` 已下载。
- `payApi` 环境变量已填写。
- `npm run deploy:cloud` 已执行。
- `npm run db:init` 已执行。
- `0.01` 元支付已成功。
- `0.01` 元退款已成功。
