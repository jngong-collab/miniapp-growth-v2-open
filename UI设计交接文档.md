# 浴小主 · 小儿推拿门店拓客小程序 — 页面功能清单

> **项目类型**：微信小程序（750rpx 设计稿宽度）
> **目标用户**：25-40 岁宝妈
> **行业**：小儿推拿 / 母婴健康
> **项目定位**：这是一个拓客裂变系统，不是普通商城。核心玩法：用户购买 → 分享链接 → 好友购买 → 邀请人获得现金返现。
> **底部 TabBar**：4 个 Tab — 首页 / AI看舌象 / 商城 / 我的

---

## Page 1：首页

**TabBar 第 1 个 Tab**

功能元素：
- 根据时间段显示问候语（早上好/下午好/晚上好）
- 显示门店名称
- 门店地址（可点击，打开地图导航）
- 三个功能入口卡片：「AI看舌象」「套餐查询」「幸运大抽奖」
- 裂变活动推广条：商品名、活动价、返现金额、已参与人数、进度条
- 技师介绍：头像、姓名、认证标识、从业年限、服务人次、好评率、个人简介、擅长领域标签
- 推荐套餐卡片：图片、名称、功效、现价、原价
- 精选商品列表（最多4个）：图片、名称、价格、折扣标签
- 进入页面时有加载骨架屏
- 无商品时显示空态提示

点击跳转：
- AI看舌象 → 舌诊页
- 套餐查询 → 套餐核销页
- 幸运大抽奖 → 抽奖页
- 活动推广条 → 裂变活动页
- 商品/套餐卡片 → 商品详情页

---

## Page 2：AI 看舌象

**TabBar 第 2 个 Tab**

此页面有 3 个状态切换：填写表单 → 照片预览 → AI分析中

**状态一：填写表单**
- 页面标题和功能说明
- 宝宝年龄选择（下拉：0-1岁/1-3岁/3-6岁/6岁以上）
- 宝宝性别选择（男宝/女宝）
- 症状描述输入框
- 拍照指南提示（3条：光线充足、舌头伸出、不加滤镜）
- 「拍舌象」按钮（调起相机或相册）
- 「查看历史分析」文字链接（显示历史次数）

**状态二：照片预览**
- 显示拍好的舌象照片
- 「重新拍摄」按钮
- 「开始AI分析」按钮

**状态三：分析中**
- 加载动画（等待约10-30秒，需要让用户感觉AI在认真工作）
- 提示文字：正在分析中

---

## Page 3：商城

**TabBar 第 3 个 Tab**

功能元素：
- 搜索框（搜索商品名称）
- 分类Tab切换：全部推荐 / 推拿服务 / 周期套餐 / 甄选产品
- 裂变活动轮播区（如有活动）：商品图片、活动标签、商品名、活动价、返现金额、进度条、参与人数
- 商品列表：商品图片、类型标签（产品/服务/套餐）、商品名、功效说明、现价、原价、购买按钮
- 有裂变活动的商品额外显示返现金额
- 列表底部提示「已经到底啦」
- 无商品时显示空态

点击跳转：
- 商品卡片 → 商品详情页
- 活动轮播 → 裂变活动页

---

## Page 4：我的

**TabBar 第 4 个 Tab**

功能元素：
- 用户信息区：头像、昵称、会员等级、账户余额
- 数据统计栏：总订单数、有效套餐数、邀请好友数
- 我的订单入口（4个）：待支付、已支付、去核销、退款
- 功能菜单：
  - 分享赚钱（显示已赚金额）
  - AI舌象记录（显示分析次数）
- 门店服务：
  - 联系门店（拨打电话）
  - 门店导航（打开地图）
- 管理入口（仅管理员/员工可见）：
  - 核销中心（需有核销权限）
  - 员工管理（仅管理员）
- 底部显示版本号

点击跳转：
- 订单相关 → 订单列表（按状态筛选）
- 有效套餐 → 套餐核销页
- 分享赚钱/邀请好友 → 裂变活动页
- AI舌象记录 → 舌象报告历史
- 核销中心 → 核销中心页
- 员工管理 → 员工管理页

---

## Page 5：商品详情

功能元素：
- 商品图片轮播（多张，自动播放）
- 价格区：现价、原价、折扣标签、已售数量
- 商品信息：类型标签（产品/服务/套餐）、商品名、描述、功效
- 裂变活动提示条（如有活动）：活动说明、每单返现金额，可点击查看详情
- 套餐内容（仅套餐类型）：包含的服务项列表及各项次数、有效期
- 服务保障：品质保障、配送方式、售后无忧
- 底部操作栏（固定）：分享按钮、回商城按钮、「立即购买」按钮（显示价格）
- 购买弹窗（从底部弹出）：
  - 商品缩略图、名称、单价
  - 数量选择（+/-），活动商品锁定为1件
  - 合计金额
  - 「确认购买」按钮 → 调起微信支付

点击跳转：
- 裂变活动提示 → 裂变活动页
- 回商城 → 商城Tab
- 支付成功 → 套餐核销页（服务/套餐类商品）

---

## Page 6：裂变活动

功能元素（每个活动包含以下内容）：
- 商品封面大图
- 价格展示：活动价、原价、推荐返现金额（每成功一单返多少）
- 参与进度：已参与人数、剩余份数、进度条
- 「立即购买」按钮
- 产品介绍：功效标签、产品描述
- 裂变玩法说明（3步流程）：购买活动产品 → 分享给好友 → 好友购买即返现
- 我的收益（已有数据时）：邀请好友数、累计收益、账户余额
- 邀请记录列表：被邀请人头像、昵称、时间、返现金额
- 活动规则（4条）
- 无活动时显示空态

---

## Page 7：套餐核销

有两个 Tab 切换：「我的套餐」和「出示核销码」

**Tab：我的套餐**
- 套餐卡片列表，每个显示：
  - 类型（单次服务/套餐）、状态（有效/已核销）
  - 商品名
  - 套餐各服务项的剩余次数（用进度圆点表示已用/剩余）
  - 核销码
  - 「出示核销码」按钮
- 无套餐时显示空态 + 跳转商城按钮

**Tab：出示核销码**
- 所选套餐名称和状态
- 大号条形码/二维码（给店员扫码用，要大而清晰）
- 服务选择器（套餐类型时）：选择要核销的具体服务项，每项显示剩余次数
- 警告提示：核销后不可撤回，请确认后交由工作人员操作
- 服务选择弹窗：列出各服务项及剩余次数，次数为0的禁用

---

## Page 8：核销中心（员工端）

仅管理员和有核销权限的员工可进入。

功能元素：
- 页面标题和说明
- 核销码输入框（8位数字）
- 「查询」按钮（输入满6位后激活）
- 查询结果卡片：商品名、类型标签
- 服务选择（套餐类型时）：可选的服务项列表，每项显示剩余次数
- 「确认核销」按钮
- 「重新输入」按钮
- 核销结果显示：成功/失败状态 + 消息 + 「继续核销」按钮
- 处理中的加载遮罩

---

## Page 9：员工管理（管理员端）

仅管理员可进入。

功能元素：
- 页面标题 + 员工总数
- 「添加员工」按钮
- 员工列表，每个显示：
  - 头像、姓名、手机号
  - 权限标签列表
  - 「编辑权限」按钮、「移除」按钮
- 无员工时显示空态

- 添加员工弹窗：
  - OpenID 输入（必填）
  - 姓名输入（选填）
  - 手机号输入（选填）
  - 权限勾选（多选）：核销权限、查看订单、查看数据、管理商品、管理活动
  - 「取消」「确认添加」按钮

- 编辑权限弹窗：
  - 显示员工姓名
  - 权限勾选（回显当前权限）
  - 「取消」「保存权限」按钮

---

## Page 10：舌象报告

有两个模式：报告详情 / 历史记录

**模式一：报告详情**
- 标题 + 分析日期
- 舌象照片（大图）
- AI分析结论（1-2句话）
- 舌象特征（4项）：舌色、舌苔、舌形、润燥
- 分析详情（长文本段落）
- 推荐商品列表：商品图、推荐理由、商品名、价格、原价、「了解详情」按钮 → 跳转商品详情页
- 调理建议（编号列表）
- 免责声明（小字）
- 底部操作栏：「重新分析」按钮、「分享给好友」按钮

**模式二：历史记录**
- 历史列表：缩略图、报告名称、日期，点击查看对应报告
- 无记录时显示空态

---

## Page 11：幸运大抽奖

功能元素：
- 页面标题 + 说明文字
- 剩余抽奖次数显示（每日3次）
- 九宫格抽奖盘（3×3）：
  - 外围8格为奖品格：各有图标和奖品名
  - 中心格为「GO」抽奖按钮
  - 抽奖时高亮格子依次跑灯旋转
- 活动说明（4条规则）：每日3次、到店领取、优惠券30天有效、解释权归门店
- 中奖结果弹窗：
  - 中奖：奖品图标、「恭喜中奖」、奖品名、领取提示
  - 未中奖：「再接再厉」、鼓励文字
  - 「知道了」关闭按钮
