# Tab 栏图标说明

小程序 Tab 栏需要 8 个图标文件（4 个默认态 + 4 个选中态）：

| 文件名 | 说明 | 尺寸 |
|---|---|---|
| `home.png` | 首页图标 | 81×81 px |
| `home-active.png` | 首页图标（选中） | 81×81 px |
| `tongue.png` | AI 看舌象图标 | 81×81 px |
| `tongue-active.png` | AI 看舌象图标（选中） | 81×81 px |
| `mall.png` | 商城图标 | 81×81 px |
| `mall-active.png` | 商城图标（选中） | 81×81 px |
| `profile.png` | 我的图标 | 81×81 px |
| `profile-active.png` | 我的图标（选中） | 81×81 px |

> 默认态颜色：#999999
> 选中态颜色：#FF6B35（主题橙）
> 格式：PNG，透明背景
> 建议使用线性图标风格

在微信开发者工具中打开项目前，请将对应图标放入此目录。
可使用 iconfont.cn 或 iconpark.com 下载。
